RoboClaw: An Agentic Framework for Scalable Long-Horizon Robotic Tasks

Ruiying Li, Yunlang Zhou, YuYao Zhu, Kylin Chen, Sukai Wang, Kongtao Hu, Minhui Yu, Bowen Jiang, Jiayao Ma, Zhan Su, XinHe, Yongjian Shen, yangyang, Guanghui Ren, Maoqing Yao, Wenhao Wang, Yao Mu
Paper Supplementary Code arXiv

Abstract

Vision-Language-Action (VLA) systems have shown strong potential for language-driven robotic manipulation. However, scaling them to long-horizon tasks remains challenging. Existing pipelines typically separate data collection, skill learning, and deployment, resulting in heavy reliance on manual environment resets and brittle multi-skill execution.

We present RoboClaw, an agentic robotics framework that unifies data collection, skill learning, and task execution under a single VLM-driven controller. At the skill level, RoboClaw introduces Entangled Action Pairs (EAP), which couple forward manipulation behaviors with inverse recovery actions to form self-resetting loops for autonomous data collection. This mechanism enables continuous on-policy data acquisition and iterative skill refinement with minimal human intervention.

During deployment, the same agent performs high-level reasoning and dynamically orchestrates learned skill primitives to accomplish long-horizon tasks. By maintaining consistent contextual semantics across collection and execution, RoboClaw reduces mismatch between the two phases and improves multi-skill robustness. Experiments in real-world manipulation tasks demonstrate improved stability and scalability compared to conventional open-loop pipelines, while significantly reducing human effort throughout the robot lifecycle.

Overview of RoboClaw

A VLM-based agent drives a closed-loop data engine: it schedules forward/inverse skills as Entangled Action Pairs (EAPs) for self-resetting data generation, manages multimodal context for reliable task progression, and performs automated cleaning and representation-space distribution checks. The curated trajectories are used to train independent skill checkpoints, which are then composed by the same agent at deployment time for long-horizon manipulation.

Roboclaw Autonomous Data Collection Workflow

This diagram illustrates the process of the agent interacting with a user to initiate a data collection task for the robot ("place the primer into the drawer"). The agent autonomously processes visual observations using MCP tools, evaluates the initial state of the environment, and formulates a task plan. Subsequently, it continuously executes a forward-backward operational loop (i.e., placing the item into the drawer and then taking it out) while monitoring for anomalies in real-time during execution, thereby continuously acquiring the robotic manipulation dataset.

Overview of the RoboClaw agentic framework

A Vision-Language Model (VLM) acts as a meta-controller operating under an in-context learning paradigm. Multimodal observations are integrated with structured memory (role identity, task-level memory, and working memory) to form the decision context. Through chain-of-thought (CoT) reasoning, the agent generates high-level decisions and invokes tools through a unified MCP execution interface. The same agent core governs both data collection and deployment, ensuring consistent control semantics across the full system lifecycle.
